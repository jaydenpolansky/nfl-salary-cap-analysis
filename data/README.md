# Data Files

This folder should contain the following CSV files:



## team_cap_2011_2024.csv

Salary cap data by team and year. Can be generated by running:

```r
source("scripts/scrape_spotrac.R")
```

**Columns:**
- `Year`: NFL season year
- `Team`: Team abbreviation (e.g., "KC", "PHI")
- `Total_Cap`: Total salary cap allocations
- `Cap_Space`: Available cap space
- `Active`: Active 53-man roster spending
- `Reserves`: Reserve list allocations (IR/PUP/NFI/SUSP)
- `Dead`: Dead cap money



## team_data_2011_2024.csv

Team performance data including wins, losses, and points.

**Required columns:**
- `Year`: NFL season year
- `Team`: Full team name (e.g., "Kansas City Chiefs")
- `W`: Wins
- `L`: Losses
- `Win_perc`: Win percentage (0-1)
- `PF`: Points scored (Points For)
- `PA`: Points allowed (Points Against)

This data can be compiled from NFL standings or obtained from sources like Pro Football Reference.
